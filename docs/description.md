# Project Description

## Recipe Finder & Collection Manager

### Overview

Recipe Finder & Collection Manager is a web application designed to help users discover recipes, organize their favorite dishes, and plan weekly meals efficiently.

Many people often have ingredients at home but do not know what to cook. This can lead to wasted food and time spent searching through multiple apps. Our application solves this problem by combining recipe discovery, meal planning, and shopping list management into a single platform.

Users can search recipes by dish name or by ingredients they already have. They can save interesting recipes to a personal wishlist, organize meals for the week using a meal planner, and automatically generate a shopping list based on selected recipes.

### Main Features

* **Recipe Search:** Users can search recipes by name or ingredients.
* **Recipe Details:** View cooking instructions, ingredients, nutrition information, and preparation time.
* **Wishlist:** Save favorite recipes for future cooking.
* **Meal Planner:** Plan meals for each day of the week.
* **Shopping List:** Automatically generated from planned meals.
* **Culinary News:** Displays latest food-related news articles.
* **User Authentication:** Secure login using Google OAuth2.

### Technologies Used

**Frontend**

* ReactJS
* Tailwind CSS

**Backend**

* Node.js
* Express.js
* JWT authentication

**Database**

* MySQL

**External APIs**

* Spoonacular API for recipe and ingredient data
* NewsAPI for culinary news
* Google OAuth2 for user authentication

### Project Goal

The goal of this project is to create a simple and efficient platform that helps users reduce food waste, organize meals more easily, and discover new recipes in one convenient place.
